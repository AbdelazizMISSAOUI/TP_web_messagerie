# TP Web – Application de Messagerie

## 🌐 Lien du sit web déployé avec Render
https://tp-web-messagerie-1.onrender.com/

## Description
Application web de messagerie simple (frontend + backend).
Fonctionnalités :
- afficher des messages
- ajouter un message
- supprimer un message
- changer le thème

## Architecture
TP1-web/
├── frontend/
│   ├── index.html
│   ├── script.js
│   └── style.css
├── backend/
│   ├── index.js
│   └── package.json

## Lancer en local

cd backend
npm install
npm start

ouvrir frontend/index.html

##  Déploiement
- Backend : Render (Web Service)
- Frontend : Render (Static Site)
